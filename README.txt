Pour lancer le serveur de BDD en ligne de commande:
- cd dans le dossier contenant hsqldb.jar
- lancer la commande : java -cp hsqldb.jar org.hsqldb.server.Server --database.0 file:firstBDD --dbname.0 firstBDD

Lancer l'application à partir de DemoApplication
